## 📸 Overview

NotchNotes turns your MacBook's notch into a productivity powerhouse. Hover over the notch, and a beautiful, glassmorphic panel drops down showing your pinned notes. 

No menubar clutter, no windows to manage. Just pure, frictionless note-taking exactly when you need it.

### ✨ Features
- **Hover to Open:** Just push your cursor to the top of your screen to drop down the panel.
- **Rich Text Editing:** Bold, italic, underline, strikethrough, text colors, highlight colors, and multi-size fonts — all fully supported.
- **Multiple Pinned Notes:** Pin multiple notes and let the Notch Panel become your quick-switcher.
- **Smart Sizing:** The panel automatically adjusts its height based on how much you've written, up to 80% of your screen height.
- **Full Screen Support:** The panel works perfectly even when you have other apps open in fullscreen mode.
- **Always There:** Auto-saves your notes instantly into a local database securely on your Mac.

## 🚀 One-Line Install

Want to install the app right now? Open your **Terminal** and run this single command:

```bash
curl -sL https://raw.githubusercontent.com/YOUR_USERNAME/NotchNotes/main/install.sh | bash
```

*(Note: Before running the above command, please replace `YOUR_USERNAME` with your actual GitHub username once you push this to GitHub. This script will download the latest release and put it straight into your Applications folder.)*

## 🛠️ Building from Source

If you prefer to build the app yourself:

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/NotchNotes.git
   cd NotchNotes
   ```
2. Build the app using our included script:
   ```bash
   ./build.sh
   ```
   *(This will compile the app and create a ready-to-use `NotchNotes.zip` file in the current directory).*
   
3. Unzip the file and drag `NotchNotes.app` into your Applications folder.
